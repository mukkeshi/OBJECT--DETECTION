# 🚀 Real-time Object Detection Web Application using YOLOv8 & Flask

An interactive web application that performs real-time object detection on uploaded images and video streams using **YOLOv8 (Nano)** and a **Flask (Python)** backend.

---

## ✨ Features
- 🔍 **Image & Video Detection:** Upload local videos/images and perform instant object identification.
- ⚡ **Live Video Streaming:** Uses OpenCV and Flask generator to stream detection frame-by-frame.
- 💻 **CPU-Optimized:** Powered by YOLOv8n, ensuring smooth execution without heavy GPU dependence.
- 🎨 **Modern Web Interface:** Clean HTML/CSS frontend communicating via REST APIs.

---

## 🛠️ Tech Stack
- **Backend:** Python, Flask, OpenCV
- **AI Model:** YOLOv8 (Ultralytics)
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
